What it is:
- Flexbox is a 1D layout model for distributing space along a row or column.

Why it matters:
- Useful for toolbars, alignment, responsive rows/columns and simple layout tasks.

Key properties / syntax:
- container: display:flex; flex-direction; justify-content; align-items; gap; flex-wrap
- item: flex-grow, flex-shrink, flex-basis, order

Interview questions:
1) Q: What does align-items do?
   A: Aligns items along the cross axis.
2) Q: How to make an item take remaining space?
   A: Use flex:1 (flex-grow:1).
3) Q: What's the difference between align-items and align-self?
   A: align-self overrides align-items for a specific item.
4) Q: How to wrap flex items?
   A: flex-wrap: wrap.
5) Q: What is gap in flexbox?
   A: Space between flex items (supported in modern browsers).

Common gotchas:
- Using flexbox for complex 2D grids—use CSS Grid instead.

Pro tips:
- Combine flex properties with min-width to build resilient responsive components.
